Expense Tracker

Developed using Python and the Tkinter library, this project provides a clean, user-friendly GUI for tracking personal expenses. The application's core is a split-screen design: a text-based output pane provides an immediate, summarized view of all expenses, while dedicated panels offer full CRUD (Create, Read, Update, Delete) functionality for expense items. Key statistical features give users clear insight into their spending, including total monthly/yearly costs and an identification of the highest-spending month and item.
